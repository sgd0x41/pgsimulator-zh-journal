# pgsimulator-zh-journal

本仓库用于归档和发布多个独立 HTML 页面。页面内容以中文为主，不针对任何特定域名或网站。所有页面均为静态 HTML，可直接在浏览器中打开查看。

## 项目简介

本项目是一个非结构化的 HTML 页面存档集合。页面主题多样，可能包含模拟器、工具、学习笔记等类型的独立页面。每个页面均为自包含的 HTML 文件，不依赖外部服务或后端。

## 目录说明

仓库根目录下包含若干 `.html` 文件，每个文件对应一个独立页面。目录结构保持扁平化，无深层嵌套。

```
.
├── index.html          # 可选的首页或导航页
├── page-example.html   # 示例页面（实际文件可能不同）
└── README.md           # 本文件
```

页面文件名通常反映其内容主题，但无统一命名规范。建议通过各页面内部的标题或说明了解其用途。

## 页面归档说明

- 每个 HTML 页面均为完整独立的文档，包含样式、脚本及内容。
- 页面之间无强关联，可单独访问。
- 部分页面可能引用外部资源（如 CDN 加载的库），但尽量保持自包含。
- 存档的页面可能处于不同完成阶段，部分页面仅为实验或演示用途。

## 维护说明

- 本仓库不定期更新，新增页面或修正现有页面。
- 不保证所有页面长期可用或兼容所有浏览器。
- 欢迎提交 Issue 或 Pull Request，但请注意本项目不针对任何具体网站或服务。
- 页面内容仅供参考和学习，请勿用于商业用途。

## 使用方式

直接通过 GitHub Pages 或本地文件系统访问各个 HTML 文件。如需部署到个人服务器，请确保静态文件服务器可正确提供 `.html` 文件。
